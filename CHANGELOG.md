# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added
- Initial release of Thred SDK
- TypeScript support with full type definitions
- Multiple build formats (UMD, CommonJS, ES modules)
- ChatGPT referral detection
- Browser fingerprinting for visitor identification
- Page view tracking
- Form submission tracking
- Lead enrichment API
- Auto-initialization support
- Programmatic API
- Debug logging mode
- Jest test suite
- ESLint and Prettier configuration
- Rollup build system
- Example HTML files
- Comprehensive documentation

### Features
- Zero external dependencies
- Privacy-focused tracking
- Automatic form detection
- MutationObserver for dynamic forms
- Global fingerprint caching
- Configurable API endpoints
- Graceful error handling
